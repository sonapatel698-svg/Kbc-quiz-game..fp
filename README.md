# KBC Quiz Game 🎮

A Kaun Banega Crorepati style quiz game built with HTML, CSS & JavaScript.

**▶️ Play Live:** https://sonapatel698-svg.github.io/Kbc-quiz-game

### Features:
- 4 Multiple choice questions with prize money
- 50-50 Lifeline 
- KBC inspired dark theme UI
- Responsive design - works on mobile

### Tech Stack:
HTML5, CSS3, Vanilla JavaScript
